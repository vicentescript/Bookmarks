# BOOKMARKS

Track the pages, keep the memories, quantify your curiosity.

A visual library manager to keep track of your books and reading progress.

## 🚀 Demo

Visita [vicentescript/Bookmarks](https://github.com/vicentescript/Bookmarks)

## 📖 Descripción

BOOKMARKS es una aplicación web que te permite gestionar tu biblioteca personal de forma visual. Incluye una sección hero asimétrica, animaciones en las carátulas de los libros y una interfaz responsive con diseño de rejilla.

## 🛠️ Tecnologías

- HTML5
- CSS3 (Grid, Flexbox, animaciones)
- JavaScript (vanilla)
